Move

Move makes writing DOM animations as easy as with CSS3 but since its all JavaScript, offers you more power and composition.

Examples

Move#set(prop, val)

Set prop to val.

move(box)
  .set('margin-left', 200)
  .end()

Move#add(prop, val)

Increment prop by val, where val is an Number (click several times).

move(box)
  .add('margin-left', 200)
  .end()

Move#sub(prop, val)

Decrement prop by val, where val is an Number (click several times).

move(box)
  .sub('margin-left', 100)
  .end()

Move#rotate(deg)

Rotate by deg.

move(box)
  .rotate(140)
  .end()

Move#duration(n)

Set animation duration to n which is a Number or a string such as "4s".

move(box)
  .set('background-color', 'blue')
  .duration('2s')
  .end()

Move#translate(x[, y])

Translate x and optionally y axis.

move(box)
  .translate(100, 20)
  .end()

Move#x(n) / Move#y(n)

Translate x or y axis. Aliased by Move#translateX(n) and Move#translateY(n).

move(box)
  .x(100)
  .y(20)
  .end()

Move#skew(x[, y])

Skew x, and optionally y. Move#skewX(n) and Move#skewY(n) are also available.

move(box)
  .skew(50)
  .end()

Move#scale(x[, y])

Scale the x, and optionally y axis. Move#scaleX(n) and Move#scaleY(n) are also available.

move(box)
  .scale(3)
  .end()

Move#end([fn])

The end() method triggers the animation to play. Optionally you can pass it a callback fn to call when complete or a duration.

move(box)
  .set('background-color', 'blue')
  .duration('1s')
  .end(function(){
    move(box)
      .set('background-color', 'aliceblue')
      .end('1s')
  })

Move#then([fn])

Creates a DeferredMove which will be run when this move completes. Optionally you can pass in a Move instance and it will be run on completion of this animation just as a DeferredMove would. then() returns the DeferredMove instance so you can configure it while then(move(el)) returns this for chaining. DeferredMove's have a special method pop() which returns its parent move incase you want to continue configuring it in a chain.

var moveBack = move(boxs[0])
  .set('background-color', 'aliceblue')
  .x(0)
  
move(boxs[0])
  .set('background-color', 'blue')
  .x(500)
  .then(moveBack)
  
move(boxs[1])
  .set('background-color', 'blue')
  .x(500)
  .scale(.5)
  .rotate(60)
    .then()
      .rotate(30)
      .scale(2)
      .set('border-radius', 5)
      .set('opacity', 0)
      .then()
        .rotate(270)
        .x(-500)
        .set('background-color', 'aliceblue')
        .set('border-radius', 0)
        .set('opacity', 1)

Move#ease(fn)

Use the given easing fn. The default function is "linear". For a complete list of available functions see

linear
in-quad
out-quad
in-out-quad
boxs.forEach(function(box){
  move(box)
    .ease(box.textContent)
    .x(400)
    .then()
      .x(-400)
})
